# バージョンチェックツール

## 必要要件
- python 3.7.10 以上

## ライブラリインストール
- HTTPリクエストモジュール
```
pip install requests
```
- HTMLスクレイピングモジュール
```
pip install beautifulsoup4
```

- windowsバージョンチェック用selenium
```
pip install selenium
```
- chrome driver
自身の使用しているバージョンに適したchromedriverを適用する  
[ダウンロードリンク](https://sites.google.com/a/chromium.org/chromedriver/downloads)

## 実行
python ./main.py
